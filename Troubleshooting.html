Easy Network Troubleshooting

1. Press `Win` + `R` on your keyboard
2. Type `cmd` on the Run window that just popped up then press `Ok`
3. Enter the following commands one by one:
ipconfig /release
ipconfig /renew
ipconfig /flushdns

1. Launch a Command Prompt window (CMD)
2. Enter the command: `ping -t website.com` (Replace website.com with a website/address you wanna check)

1. Open `Control Panel` 2. Select `Network and Internet` 3. Select `Network and Sharing Center` 4. Select `Change adapter settings` 5. Right click your connection adapter and select `Properties` 6. Locate `Internet Protocol Version 4 (TCP/IPv4)` then select `Properties` 7. Check `Use the following DNS Server Addresses:` 8. Input your preferred DNS Servers and Press `OK` 9. Locate `Internet Protocol Version 6 (TCP/IPv6)` then select `Properties` 10. Check `Use the following DNS Server Addresses:` 11. Input your preferred DNS Servers
