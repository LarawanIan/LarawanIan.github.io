1. Download and Install HandBrake from https://handbrake.fr/
2. Run HandBrake

Configuring HandBrake default settings
1. Select Preferences upon launching HandBrake (Or go to Tools -> Preferences)
2. Select Output Files
3. Set Default Path to {source_path}
4. Set File Format to {source}
5. Change MP4 File Extension to Always Use MP4

HandBrake Main Toolbar
1. In the Preset Selection Menu choose Fast 1080p30 or Fast 720p30 depending on how much storage you wanna save

HandBrake Video Tab
1. Change Framerate(FPS) to Same as Source
(If you wanna save more space set FPS to 24 or 30)
2. Set Constant Quality to 25
(Higher CQ Value = Lower Video Quality)

Handbrake Audio Tab
(It's safe to leave this tab without changing anything)
1. If you want to use the same audio set Codec to "X" Passthru
(X = The Audio codec of your video)
2. To save even more space, lower the Bitrate

Finalization
1. Set your file's name with the Save As bar located at the bottom of HandBrake
2. When you are satisfied with the settings click on Start Encode (The one with a green play button)

Searching for torrents within qBittorrent

3. Scroll down and click on Releases
4. On the latest version, click on Assets
5. Download and then install the latest app-release.apk file
6. Launch the Obtainium app
7. Click on Add App
8. Enter this link in the App Source URL: github.com/TeamVanced/VancedMicroG
9. Click on Add
10. Android will ask for your prompt to allow Obtainium to install applications, make sure to allow it
11. Go back to the Obtainium app and wait for the download to finish
12. You will be prompted to install Vanced MicroG, click on Install
13. Go back to Obtainium's Add App page, and then enter this link in the App Source URL: github.com/LarawanIan/ian-rvx
14. Wait for the download to finish
15. You will be prompted to install ReVanced Extended, click on Install
16. Go back to Obtainium's Add App page, and then enter this link in the App Source URL: github.com/JunkFood02/Seal
17. Select the compatible APK for your device and wait for the download to finish
18. You will be prompted to install Seal, click on Install
19. Obtainium will now keep track of those apps' updates!

Integrating Seal into YouTube ReVanced
1. Open YouTube ReVanced
2. Go to the settings page
3. Scroll down and click on ReVanced Settings
4. Click on Overlay Button Settings
5. Make sure Show download button is enabled
6. Now click on Downloader Settings
7. Click on Downloader package name
8. In the text field enter: com.junkfood.seal



1. Open Seal
2. Open Seal's Settings page
3. Select Network
4. Select Cookies
5. Click on Generate new cookies
6. Enter the URL of the site you want to have cookies for
EXAMPLE: Instagram: https://instagram.com | YouTube: https://youtube.com
It's important to have "https://"
7. Click on Generate new cookies
8. Seal's built in browser will open the URL you entered
9. Login to your account as you would normally in Seal's web browser
10. After logging in, press Done at the top right of the browser
